### Dagger2

------

- 가장 많이 보이는 DI Framework이다.
- Dagger2 적용을 위해선 환경을 세팅해야 하는 과정과 원활한 적용에 필요한 러닝커브가 크다.
- 러닝커브에 사용되는 비용이 상대적으로 저렴한 (부담이 적은) 사람들이 쓰기에 적합하다.
