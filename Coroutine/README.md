## 코루틴(Coroutine) 이란?

코루틴은 안드로이드에서 백그라운드 스레드에서 코드를 처리할 때 사용하는 하나의 방법이다.

코루틴을 사용하는 대표적인 경우

- 네트워크 리퀘스트 (Retrofit, Volley 등)
- 내부 저장소 접근 (Room SQLite 등)

코루틴은 코드가 간결하고, 블록을 처리할 수 있기 때문에 하나의 통신 후에 또 다른 시퀸스 작업에 최적화 되어 있다.
